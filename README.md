📚 Study Tracker AI
Çalışma kayıtlarını tutan, analiz eden ve akıllı öneriler sunan terminal tabanlı Python uygulaması.

🚀 Özellikler

Çalışma kaydı ekleme (ders, süre, tarih, verim puanı)
Kayıtları JSON dosyasına otomatik kaydetme
Tüm kayıtları listeleme
Toplam süre, ortalama verim ve en çok çalışılan ders analizi
Ders bazlı ve gün bazlı detaylı analiz
Matplotlib ile görsel grafikler
Kural tabanlı akıllı öneri sistemi


🛠️ Kullanılan Teknolojiler

Python 3
json (veri saklama)
datetime (tarih işlemleri)
matplotlib (grafikler)
